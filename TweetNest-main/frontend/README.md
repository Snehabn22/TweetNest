# Frontend Social Media Application

## Project Overview

This project is a frontend application for a social media platform built using React, Vite, Tailwind CSS, and DaisyUI.  It provides functionalities for user authentication (login and signup), creating posts, viewing a newsfeed, and managing user profiles.  The application utilizes placeholder images for avatars and posts, and includes loading spinners for improved user experience.

## Features

* **User Authentication:** Secure login and signup functionality.
* **Home Page:** Displays a feed of posts.
* **Post Creation:**  Allows users to create and share new posts.
* **Profile Page:** Displays user profile information and allows for editing.
* **Notification Page:**  Displays user notifications. (Functionality inferred)
* **Loading Spinners:** Provides visual feedback during loading operations.
